📥 导出

导出当前小组、成员和积分数据（含积分流水、基础信息）。JSON 适合回写 students.json；Excel 适合离线编辑后再导入。

📤 导入

支持 .json、.xlsx、.xls。全量覆盖当前小组、成员基础信息与积分数据，建议导入前先导出备份。

💾 本地自动保存

程序会把积分、小组、座位等改动自动保存到本浏览器的 localStorage，下次打开自动恢复，不再重置。 提醒：localStorage 只对本台电脑 + 本浏览器生效，换设备或清理浏览数据会丢失。建议每周额外导出一次 JSON 作为备份。
